Selected Work

Things I've Built

AI systems I built to automate demand gen work — each one replacing hours of manual effort.

Weekly Channel Analysis Engine

Claude API Python Paid Media Automation

An AI workflow that ingests raw LinkedIn, Google, and Meta performance exports, runs Python-based analysis through Claude, and delivers a structured HTML report with channel insights, budget recommendations, and anomaly flags — every week, on demand. Replaces ~4 hours of manual analyst work per cycle.

Ad & Lead Quality Analysis Pipeline

Claude API Python ICP Scoring HTML Reports

A data pipeline that scores leads from LinkedIn Ads, Google Ads, Marketo, and Salesforce against ICP criteria — bucketing them into segments like True ICP, Near-Miss, and Noise — and delivers an interactive HTML report with campaign-level quality grades. Built to answer: are we buying the right leads?

Automated A/B Testing System

Claude API CRO Landing Pages Ad Copy

A two-phase system: Phase 1 generates 50+ copy variants and scores them with a simulated expert panel before any traffic hits — taking a landing page from 62 to 87/100 in 8 minutes. Phase 2 deploys the winner and runs live traffic mutations week-over-week, compressing a 14-day A/B cycle into hours.

AI-Powered ABM Targeting Engine

Claude API ABM Account Scoring Paid Media

A system that ingests a target account list, enriches each account with firmographic and intent signals, scores them against ICP fit, and outputs a prioritized account tier list with recommended channel mix per tier. Turns a spreadsheet of accounts into a deployment-ready ABM plan in minutes.

Lifecycle Email Personalization Engine

Claude API Lifecycle Email Personalization

An AI pipeline that takes CRM segment data and generates personalized email copy variants per persona — subject lines, body, and CTA — at scale. Built to feed Marketo or HubSpot sequences with high-quality, on-brand copy without a copywriter bottleneck.
