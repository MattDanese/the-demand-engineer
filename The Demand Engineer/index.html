Weekly Channel Analysis Engine

Claude API LinkedIn Ads API Google Ads API Slack Node.js

A Claude Code-powered system that connects the LinkedIn Ads API, Google Ads API, and Anthropic API to eliminate manual reporting entirely. Pulls prior-week data every Monday at 8AM, runs Claude analysis with a locked senior analyst persona, and posts a structured briefing to Slack — headline story, what worked, what needs attention, and 3 prioritized actions. Saves an estimated 12–18 hours of analyst time per week across 12 discrete automations including anomaly detection, budget pacing, and a real-time Slack Q&A bot.

Ad & Lead Quality Analysis Pipeline

Claude API Python ICP Scoring Marketo Salesforce

An on-demand diagnostic system that accepts raw CSV and XLSX exports from LinkedIn Ads, Google Ads, Marketo, and Salesforce — runs Python-based processing through Claude — and delivers interactive HTML reports with ICP scoring and campaign quality grades. Scores every lead against ICP criteria (Title, Function, Company Size) and buckets them into True ICP, Near-Miss, Wrong Title, and Noise. Built to answer one question: are we buying the right leads?

Automated A/B Testing System

Claude API CRO Landing Pages Ad Copy Autoresearch

A two-phase system that eliminates the 14-day testing bottleneck. Phase 1 (Autoresearch) generates 50+ copy variants and scores them with a simulated 5-person expert panel — CMO, Skeptical Buyer, CRO Specialist, Senior Copywriter, ROI-Focused CEO — before any traffic hits the page. Took a landing page from 62 to 87/100 across 51 variants in 8 minutes. Phase 2 (Autogrowth) deploys the winner and runs live traffic mutations week-over-week, compressing a 14-day A/B cycle into hours.

Claude API Brand Strategy ABM Pipeline Prompt Engineering

Marketo Audit & RevOps Infrastructure

Claude API Marketo RevOps Python Sales Enablement

A diagnostic and documentation system at the intersection of paid media, marketing ops, and sales. Accepts Marketo activity log exports, runs Python parsing through Claude, and diagnoses record-by-record why leads failed to enter nurture programs — identifying blockers like Master Send List exclusions, stale unsubscribes, and EMEA routing gaps.
